PROJECT OVERVIEW
The Final Project is a group-based system integration project that requires students to design, develop, and present a real-world enterprise integration solution aligned with the course topics.

Students are expected to apply concepts such as:

ERP Integration
APIs and Middleware
Enterprise Application Integration (EAI)
Integration Testing
Security in Cloud-Based Systems
DevOps (CI/CD, Monitoring, Observability)
The project must demonstrate how multiple systems interact to form a secure, scalable, and efficient integrated solution.

PROJECT OBJECTIVES
By completing this project, students should be able to:

Design a system using ERP strategies, APIs, and middleware
Implement integration workflows across systems
Apply testing, security, and DevOps practices
Present a complete system architecture with real-world relevance
